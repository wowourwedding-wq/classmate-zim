ℹ️ How does this work? · First-time guide

Step 1 — Add parent contact details

  1. Go to 👥 Class List
  2. Tap + Add learner for each pupil
  3. Fill in Parent / guardian name and Parent phone — these are the contacts used here
  4. Phone format: +263 77 123 4567 or just 077 123 4567. Both work — we normalise automatically.

Step 2 — Write your message

Use placeholders so each parent gets a personalised message:

  • {parent} → "Mrs Moyo"
  • {child} → "Tendai"
  • {teacher} → your name
  • {school} → your school's name
  • {date} → today's date

Example: "Good day {parent}, just a reminder {child} has a Maths test tomorrow. — {teacher}"

Step 3 — Pick recipients

By default everyone is selected. Tick / untick individuals to narrow it down. School Edition gives a "Whole school" source pulling from the School Directory instead of just one class.

Step 4 — Send

  1. 📱 WhatsApp links — your phone's WhatsApp opens with each parent's message ready. Tap send. ~2 seconds per parent.
  2. 📋 Copy SMS — copies phone, parent, message for every recipient to your clipboard. Paste into BulkSMS.co.zw, EcoCash bulk, Africa's Talking, or your school's gateway.
  3. 📋 Copy numbers — comma-separated phones only, for quick paste.
  4. ⬇️ Export CSV — full download for school records.

Step 5 — Save it as a template

Tap 💾 Save as template so you can re-use the message next week (parent meeting, fees reminder, sports day, etc.). Templates appear at the bottom of this page.

💾 Back up weekly — important

Everything you enter is stored on this device only — nothing is sent anywhere. If you lose your phone, change browser, or uninstall the app without a backup, the data is gone forever. Make a backup a Sunday-evening habit.

📥 How to back up (30 seconds, once a week)
  1. Tap the Download backup button below
  2. Your phone saves a file named classmate-zim-backup-YYYY-MM-DD.json
  3. Email it to yourself (subject: "ClassMate backup") or save to Google Drive / WhatsApp's "Saved Messages"
  4. Done — that's your safety net for the week
📤 How to restore (after losing your phone)
  1. On the new device, open classmate-zim.netlify.app/app.html
  2. Tap Open my planner on the welcome screen
  3. Sidebar → ⚙️ Settings → the Choose file button below
  4. Pick the latest classmate-zim-backup-*.json file
  5. Confirm — everything is back in 2 seconds
